# 🥋 Dojo Manager

Gestionale completo per dojo di karate — portale atleti + pannello admin.

## Tecnologie
- React + Vite
- Supabase (database + autenticazione)
- Vercel (hosting)

## Configurazione
Crea un file `.env.local` con:
```
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
VITE_ADMIN_EMAIL=...
```
